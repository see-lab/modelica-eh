# Modelica Energy Hub (EH) Package

The Modelica `EnergyHub` (EH) package is a free modeling
repository for designing and optimizing multi-energy systems and energy hubs. 
THe package includes both linear and nonlinear formulations for validation and 
use.


Many models are based on models from the
[Modelica Buildings Library](https://simulationresearch.lbl.gov/modelica/).
We aim to use the same general modeling
structure and practices as detailed in [the MBL Best Practices](https://simulationresearch.lbl.gov/modelica/userGuide/bestPractice.html).

## Dependencies
This library uses Modelica v4.0.0 and Buildings v11.0.0. 

## Tool Compatibility
These models have have been tested with Dymola 2025x and OpenModelica 1.25.0.
